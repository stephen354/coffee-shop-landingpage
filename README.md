Landing page elegan bertema coffee shop, dibuat dengan Next.js 16, React 19, dan Tailwind CSS 4.

| Kategori              | Teknologi             | Deskripsi                                                         |
| --------------------- | --------------------- | ----------------------------------------------------------------- |
| **Framework**         | **Next.js 16**        | Framework React modern untuk pengembangan web cepat dan efisien.  |
| **Library UI**        | **React 19**          | Library JavaScript deklaratif untuk membangun antarmuka pengguna. |
| **Styling**           | **Tailwind CSS 4**    | Framework CSS berbasis utility untuk desain cepat dan konsisten.  |
| **Animation**         | **Framer Motion**     | Library animasi untuk komponen React dengan performa tinggi.      |
|                       | **GSAP (GreenSock)**  | Library animasi profesional untuk efek transisi kompleks.         |
|                       | **Split Type**        | Memecah teks menjadi karakter/kata/baris untuk animasi tipografi. |
| **Scrolling Effect**  | **Locomotive Scroll** | Efek parallax dan smooth scrolling yang halus.                    |
| **Slider / Carousel** | **Swiper.js**         | Komponen slider responsif dan interaktif.                         |
| **Icons**             | **React Icons**       | Koleksi ikon SVG ringan untuk React.                              |
| **Linting**           | **ESLint 9**          | Linter untuk menjaga kualitas da                                  |
